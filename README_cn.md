# StarPage

一个视觉效果出色的单文件浏览器起始页，拥有动态星空背景和快捷链接。

## 功能特性

- **动态 Canvas 背景** — 光球、网络节点、粒子、流星、极光丝带、萤火虫、六边形网格、电路纹路等，全部实时渲染并跟随鼠标交互。
- **实时时钟** — 支持 12 小时制和 24 小时制切换，点击右上角按钮即可切换。
- **多引擎搜索** — 支持 Google、Bing、百度三种搜索引擎，按 `Enter` 键搜索。
- **快捷链接** — 可自定义的快捷方式卡片网格，数据保存在 `localStorage` 中。
- **添加 / 编辑 / 删除** — 点击 `+` 按钮添加链接，右键点击卡片编辑，悬停后点击 `x` 删除。
- **暖色琥珀主题** — 统一的暖色调配色，搭配暗角、胶片颗粒和毛玻璃效果。
- **零依赖** — 纯 HTML + CSS + JavaScript 单文件，无需任何构建工具。

## 快速开始

1. 克隆或下载本仓库。
2. 在浏览器中打开 `star-page.html`。
3. （可选）将其设置为浏览器主页或新标签页。

## 使用方法

| 操作 | 方式 |
|---|---|
| 搜索 | 在搜索框输入内容，按 `Enter` |
| 切换搜索引擎 | 点击 `Google` / `Bing` / `Baidu` 标签 |
| 切换 12/24 小时制 | 点击右上角时钟图标 |
| 添加链接 | 点击 `+` 卡片 |
| 编辑链接 | 右键点击卡片 |
| 删除链接 | 悬停卡片后点击 `x` 按钮 |
