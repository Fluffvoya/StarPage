# StarPage

A visually stunning, single-file browser start page with an animated cosmic background and quick-access shortcuts.

## Features

- **Animated Canvas Background** — Orbs, network nodes, particles, shooting stars, aurora ribbons, fireflies, hex grids, circuit traces, and more. All rendered in real-time and reactive to your mouse cursor.
- **Live Clock** — Supports 12-hour and 24-hour formats with a toggle button in the top-right corner.
- **Multi-Engine Search** — Switch between Google, Bing, and Baidu. Press `Enter` to search.
- **Quick Links** — A customizable grid of shortcut cards. Data is persisted in `localStorage`.
- **Add / Edit / Delete** — Click the `+` button to add a link, right-click a card to edit, hover and click `x` to delete.
- **Warm Amber Theme** — Cohesive warm color palette with subtle vignette, film grain, and glassmorphism effects.
- **Zero Dependencies** — Pure HTML + CSS + JavaScript in a single file. No build tools required.

## Getting Started

1. Clone or download the repository.
2. Open `star-page.html` in your browser.
3. (Optional) Set it as your browser's home page or new tab page.

## Usage

| Action | How |
|---|---|
| Search | Type in the search box, press `Enter` |
| Switch search engine | Click `Google` / `Bing` / `Baidu` tabs |
| Toggle 12/24h | Click the clock icon in the top-right |
| Add a link | Click the `+` card |
| Edit a link | Right-click on a card |
| Delete a link | Hover a card, click the `x` button |
